# Requirements: React + GSAP Todo App

## Functional Requirements
1. **React State**: Persistent state using `useState` and `useEffect`.
2. **GSAP Animations**:
    - Staggered entrance for list items.
    - Elastic addition animation.
    - Fade-and-slide deletion.
3. **Optimistic UI**: React-native responsiveness.

## Non-Functional Requirements
- **Mobile First**: Fully responsive layout.
- **Aesthetics**: Dark mode with neon accents (React blue + GSAP green).
- **Architecture**: Modular component structure (`TodoItem`, `AddTodo`, `Filters`).

## Technical Constraints
- Vite based.
- React 18+.
- GSAP 3+.
