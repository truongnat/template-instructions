# React + GSAP Premium Todo App

A state-of-the-art Todo application demonstrating the integration of **React** for state management and **GSAP** for high-performance animations.

## Key Features
- **GSAP Timelines**: Smooth staggered entrance animations for the entire list.
- **Elastic Transitions**: Adding tasks uses an elastic bounce effect.
- **Micro-animations**: Hover effects on interactive elements with subtle scaling.
- **Optimistic Deletion**: Animation triggers before the state update for a seamless feel.
- **Clean Architecture**: Decoupled styles (CSS) and logic (React + Hooks).

## Tech Stack
- **Framework**: [React](https://react.dev/)
- **Animation**: [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/)
- **Bundler**: Vite

## Getting Started

### Installation
```bash
cd demo/todo-app-react
npm install
```

### Development
```bash
npm run dev
```
