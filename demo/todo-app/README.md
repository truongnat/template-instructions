# Premium Todo App Demo

A high-performance, visually stunning Todo application built with Vanilla JavaScript and CSS, bundled with Vite. This project demonstrates the capabilities of the **Agentic SDLC Framework**.

## Features
- **Modern UI**: Smooth gradients, glassmorphism, and responsive design.
- **Micro-animations**: Seamless transitions for task management.
- **Clean Architecture**: Structured into Domain, Use Case, and View layers.
- **Optimistic UI**: Instant feedback on user actions.
- **Persistence**: Remembers your tasks using LocalStorage.

## Tech Stack
- **Bundler**: Vite
- **Logic**: Vanilla JavaScript (ES6+)
- **Styling**: Vanilla CSS (Custom tokens)

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
```bash
cd demo/todo-app
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```
