# Requirements: Premium Todo App

## Functional Requirements
1. **Task Management**: CRUD operations for tasks.
2. **Filtering**: All, Active, Completed filters.
3. **Persistence**: LocalStorage support.
4. **Empty State**: Graceful placeholder UI.

## Non-Functional Requirements
- **Visuals**: HSL-based palette, gradients, glassmorphism.
- **Interactions**: Micro-animations and optimistic UI updates.
- **Performance**: Instant interaction speeds.

## Technical Constraints
- No external heavy frameworks.
- Vanilla JS, CSS, and HTML.
- Vite for bundling.
