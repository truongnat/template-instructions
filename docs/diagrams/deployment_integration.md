## API Integration Patterns

```
┌─────────────────────────────────────────────────────────────────┐
│                    API INTEGRATION PATTERNS                     │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│  1. SYNCHRONOUS REQUEST-RESPONSE                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Client                                                         │
│    │                                                            │
│    ├─ POST /api/workflows/execute ──────────────────────────┐  │
│    │                                                        │  │
│    │  Agentic SDLC                                         │  │
│    │  ├─ Parse request                                    │  │
│    │  ├─ Validate workflow                                │  │
│    │  ├─ Execute workflow                                 │  │
│    │  ├─ Aggregate results                                │  │
│    │  └─ Return response                                  │  │
│    │                                                        │  │
│    └─ 200 OK + Results ────────────────────────────────────┘  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│  2. ASYNCHRONOUS JOB SUBMISSION                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Client                                                         │
│    │                                                            │
│    ├─ POST /api/workflows/submit ───────────────────────────┐  │
│    │                                                        │  │
│    │  Agentic SDLC                                         │  │
│    │  ├─ Create job                                        │  │
│    │  ├─ Queue workflow                                    │  │
│    │  └─ Return job_id                                     │  │
│    │                                                        │  │
│    └─ 202 Accepted + job_id ──────────────────────────────┘  │
│    │                                                            │
│    ├─ GET /api/jobs/{job_id} ──────────────────────────────┐  │
│    │                                                        │  │
│    │  Agentic SDLC                                         │  │
│    │  ├─ Lookup job                                        │  │
│    │  ├─ Return status                                     │  │
│    │  └─ Return results (if complete)                      │  │
│    │                                                        │  │
│    └─ 200 OK + Status/Results ────────────────────────────┘  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│  3. WEBHOOK CALLBACKS                                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Client                                                         │
│    │                                                            │
│    ├─ POST /api/workflows/execute ───────────────────────────┐ │
│    │   (with webhook_url)                                  │ │
│    │                                                        │ │
│    │  Agentic SDLC                                         │ │
│    │  ├─ Queue workflow                                    │ │
│    │  └─ Return job_id                                     │ │
│    │                                                        │ │
│    └─ 202 Accepted ────────────────────────────────────────┘ │
│                                                                 │
│    [Workflow executes in background]                           │
│                                                                 │
│    Agentic SDLC                                                │
│    │                                                            │
│    └─ POST {webhook_url} ──────────────────────────────────┐  │
│       (with results)                                       │  │
│                                                            │  │
│       Client                                               │  │
│       ├─ Receive webhook                                  │  │
│       ├─ Process results                                  │  │
│       └─ Return 200 OK                                    │  │
│                                                            │  │
│       Agentic SDLC                                         │  │
│       └─ Confirm delivery                                 │  │
│                                                            │  │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│  4. STREAMING RESPONSES                                         │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  Client                                                         │
│    │                                                            │
│    ├─ GET /api/workflows/execute/stream ────────────────────┐ │
│    │                                                        │ │
│    │  Agentic SDLC                                         │ │
│    │  ├─ Start workflow                                    │ │
│    │  ├─ Stream step results                               │ │
│    │  ├─ Stream agent outputs                              │ │
│    │  ├─ Stream learning events                            │ │
│    │  └─ Stream completion                                 │ │
│    │                                                        │ │
│    └─ 200 OK + Stream ────────────────────────────────────┘  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```
