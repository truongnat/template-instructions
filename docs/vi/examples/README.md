# Ví Dụ Code (Code Examples)

**Phiên bản**: 3.0.0  
**Cập nhật lần cuối**: 11/02/2026


Thư mục này chứa các ví dụ code hoàn chỉnh và có thể chạy được cho Agentic SDLC v3.0.0.

## Cấu Trúc

### Basic Examples (Ví Dụ Cơ Bản)
Các ví dụ cơ bản giúp bạn bắt đầu với Agentic SDLC:
- `01-configuration.py` - Cấu hình hệ thống
- `02-simple-agent.py` - Tạo agent đơn giản
- `03-basic-workflow.py` - Workflow cơ bản
- `04-error-handling.py` - Xử lý lỗi

### Intermediate Examples (Ví Dụ Trung Cấp)
Các ví dụ nâng cao hơn với nhiều tính năng:
- `05-multi-agent.py` - Hệ thống multi-agent
- `06-intelligence.py` - Sử dụng intelligence features
- `07-plugin-dev.py` - Phát triển plugin
- `08-cli-usage.sh` - Sử dụng CLI

### Advanced Examples (Ví Dụ Nâng Cao)
Các ví dụ phức tạp cho production:
- `09-complex-workflow.py` - Workflow phức tạp
- `10-distributed.py` - Hệ thống phân tán
- `11-integration.py` - Tích hợp external tools
- `12-production.py` - Production setup

## Yêu Cầu

Tất cả các ví dụ yêu cầu:
- Python 3.8+
- Agentic SDLC v3.0.0

Cài đặt dependencies:
```bash
pip install agentic-sdlc
```text

## Chạy Ví Dụ

Mỗi ví dụ có thể chạy độc lập:
```bash
python docs/vi/examples/basic/01-configuration.py
```

## Cấu Trúc Ví Dụ

Mỗi ví dụ bao gồm:
- **Setup Instructions**: Hướng dẫn thiết lập
- **Dependencies**: Danh sách dependencies
- **Code**: Code hoàn chỉnh với comments
- **Expected Output**: Output mong đợi
- **Notes**: Ghi chú bổ sung
