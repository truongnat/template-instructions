# Resources Directory

This directory contains package data including templates, workflows, and rules that are distributed with the SDK.

## Structure

- **templates/**: Template files for projects, workflows, and agents
  - `project/`: Project initialization templates
  - `workflow/`: Workflow definition templates
  - `agent/`: Agent configuration templates

- **workflows/**: Default workflow definitions
  - `examples/`: Example workflow configurations

- **rules/**: Default rules and guidelines
  - Coding standards
  - Review guidelines
  - Best practices

## Usage

These resources are accessible programmatically through the SDK's configuration system and are included in the package distribution.
