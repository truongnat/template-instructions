---
description: Process a natural language request using the agentic-sdlc framework
---

1. Ask the user for the task or requirement they want to implement.
2. Run the following command to process the request:
```bash
asdlc run "{{request}}"
```
3. Analyze the generated skill instructions and SDLC task plan.
4. Proceed to the next task using the `asdlc task next` workflow.
