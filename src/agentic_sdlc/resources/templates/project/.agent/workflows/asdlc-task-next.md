---
description: Get and execute the next task in the SDLC pipeline
---

1. Run the following command to get instructions for the next pending task:
```bash
asdlc task next
```
2. Follow the "SKILL INSTRUCTIONS" and "EXECUTION PROMPT" provided in the output.
3. Once the task is completed, save the output to a file.
4. Submit the task output using the `asdlc task submit` workflow.
