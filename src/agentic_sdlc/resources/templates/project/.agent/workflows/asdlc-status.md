// turbo-all

1. Run the following command to view the SDLC board:
```bash
asdlc status
```

2. The board shows all tasks organized by status: TODO, IN_PROGRESS, DONE.

3. If there are tasks in TODO, suggest running `/asdlc-task-next` to pick up the next one.
